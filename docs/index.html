Rewards Points Farmer

一款稳定、智能、安全的 Microsoft Rewards 全自动搜索挂机脚本

🚀 立即安装脚本

✨ 功能介绍

  • 全自动执行任务：一键开启，自动读取后台数据并完成每日 PC 搜索任务。
  • 深度人工模拟：内置真实打字机效果、随机页面滚动、长短随机延迟等机制，降低被检测为机器人的概率。
  • 中英双语环境支持：兼容中英文语言环境下的状态抓取，智能区分“已完成”与“未完成”任务。
  • 内嵌可视化面板：在网页内自动生成悬浮 UI（支持深色模式），实时显示当前搜索进度、倒计时与下一个任务。
  • 异常自动容错：网络中断或搜索分数未增加时，脚本会自动进入“休息模式”并择机重试，最大化保障稳定性。

📦 使用方式

第一步：安装脚本管理器

在使用本脚本前，您需要在浏览器中安装 Tampermonkey 扩展程序（油猴插件）。

  • Edge: 获取 Tampermonkey
  • Chrome: 获取 Tampermonkey

第二步：安装本脚本

点击页面顶部的“立即安装脚本”按钮，Tampermonkey 会自动弹出安装确认页面，点击“安装”即可。

第三步：开始赚取积分

1. 打开浏览器并访问 Bing.com
2. 确保您已登录您的微软账户。
3. 页面右下角将出现一个 Rewards 悬浮球，点击悬浮球并在面板中点击“▶ 开始执行”即可。
